## 概要
2Dスマホアプリの開発（甥っ子に遊んでもらう用）

※ ストアリリースまで対応、以降、更新がなかったためストアから削除

#### 技術スタック
- Unity(C#)
- Xcode

#### 内容
- ジグソーパズルとテトリスを合わせたようなゲーム
- 上から落ちてくるパズルピースを左右にスワイプ、タップで回転させて一致する箇所に当てはめる
  - 異なるピースだと、積み重ねたピースが崩れ、最初からやり直しとなる
- クリアすると次のステージが表示され、メニューから今まで遊んだステージを選択可能
